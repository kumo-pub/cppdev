# 性能

有一个 [native JSON benchmark collection][1] 项目，用于评估 37 个 JSON 库在不同操作下的速度、内存使用和代码体积。

[1]: https://github.com/miloyip/nativejson-benchmark

Merak 0.1 版本的性能测试文章可以参考 [这里](https://code.google.com/p/merak/json/wiki/Performance)。

此外，还可以参考以下第三方基准测试。

## 第三方基准测试

* [杂项 C++ JSON 解析器与生成器的基础基准测试](https://github.com/mloskot/json_benchmark) —— 作者 Mateusz Loskot（2013 年 6 月）

* [casablanca](https://casablanca.codeplex.com/)
* [json_spirit](https://github.com/cierelabs/json_spirit)
